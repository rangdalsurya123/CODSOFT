# Iris Flower Classification 

## Project Overview
This project focuses on classifying iris flowers into three species based on their physical measurements. It is a beginner-friendly machine learning project implemented using Python and scikit-learn.

### Iris Flower Species:
- Setosa
- Versicolor
- Virginica

---

## Dataset
The project uses the **Iris dataset**, which contains the following features:
- Sepal length
- Sepal width
- Petal length
- Petal width

---

## Machine Learning Model
- Algorithm: **Logistic Regression**
- Library used: **scikit-learn**
- Task: Multi-class classification
